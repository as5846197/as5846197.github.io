Hexo mac下搭建

  • hexo
  • mac


　　经常看大神们的博客，使得我写博客的欲望越来越强烈。正好看到一篇帖子是教搭建github博客的。最近公司有点忙，搭建博客只能放在晚上回家，哎，实在有点苦逼。搭建hexo的时候，也是按照别人的步骤操作的，但是路上还是遇到了不少坑。但是最后还是成功搭建。嘿嘿~~~，给大家吃一个定心丸，hexo搭建起来其实很简单，主要就是配置的时候需要注意一点。而且markdown语法也很简单。

正文：

　　由于我是在mac上搭建的，所以主要还是针对mac的用户，但是Windows也可以参考，我搭建过程中也参考了很多Windows下搭建的教程。

添加依赖

Node.js

　　到Node.js官网下载安装就可以,然后直接安装。

Git

　　到Git官网下载安装就可以,同样是直接安装。想要了解Git更多用法的同学点这里.

Github账户

　　到Github上注册，注册完成后，配置好自己的ssh，然后新建一个项目。命名规则：你的github账号.github.github.io，一定要这样命名这里是固定的。

Hexo 安装和配置

Hexo 安装

$ npm install -g hexo-cli

在这里可能有的时候会不让你安装，只要切换到root安装就好了。

建站


$ cd 你要设为站点的目录
$ hexo init
$ npm install

配置 _config.yml

　　配置站点根目录下的_config.yml文件。配置可以参考hexo文档，根据你的需要配置就可以了。这里唯一需要注意的是，如果用了第三方主题，小心配置时区和显示语言选项。最后还是要说一句,配置git的方法。


# Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
  type: git
  repo: https://github.com/用户名/用户名.github.io.git
  branch: master

注意：

  • 配置文件后面的冒号“：”要有一个空格

    Hexo 基本操作

    添加新文章

    hexo new “post_name”

    生成静态页面

    hexo generate 可以简写成 hexo g

    本地预览

    hexo server 可以简写成 hexo s
    这里成功后可以到浏览器上，输入网址http://localhost:4000预览。

    上传到git_hub上

    hexo deploy 可以简写成 hexo d

注意：

  • 如果出现notfound git 可以尝试 npm install hexo-deployer-git --save

至此，hexo已经搭建成功。
　　由于我是一个颜值控，所以不甘心用hexo的landscape主题。这俩天的时间主要是在找主题和配置主题。主题里面一般都有配置说明。像：多说，百度统计的配置等。这里附上知乎上面的主题推荐.

总结

Hexo的常用操作命令：


添加新文章
hexo new "post_name"
生成静态页面
hexo generate 可以简写成 hexo g
本地预览
hexo server 可以简写成 hexo s
上传到git_hub上
hexo deploy 可以简写成 hexo d

删除文章操作：
　　进入站点根目录下的source->_posts文件夹，删除不需要的文章。然后执行 hexo clean 清理数据,之后再重新生成静态页面，执行本地预览，就看以看见文章已经删除了。
